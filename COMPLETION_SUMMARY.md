## 🚀 快速开始

### 1. 初始化项目
```bash
/writer:init "我的项目"
```

### 2. 执行七阶段流程
```bash
/writer:1-research      # 内容调研
/writer:2-design        # 风格设计
/writer:3-constitution  # 创作宪法
/writer:4-outline       # 大纲编写
/writer:5-validate      # 合规性审查
/writer:6-draft         # 章节编写
/writer:7-finalize      # 最终审查
```

### 3. 查看文档
- 用户指南: `USER_GUIDE.md`
- 系统架构: `AGENTS.md`
