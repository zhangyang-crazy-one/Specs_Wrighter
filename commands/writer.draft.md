# writer.draft - 撰写文档

## 命令描述

根据大纲撰写或编辑文档内容,支持章节级别的写作。

---

## 用户输入

请提供以下信息:

1. **项目路径**: [project-path]
2. **操作类型**: 
   - write (撰写新内容)
   - edit (编辑现有内容)
3. **目标章节**: [章节编号或标题]
4. **写作指示**: [用户的具体写作要求]

---

## 执行大纲

### 步骤1: 读取项目信息

- 读取 `project-config.json`
- 读取 `outline.md`
- 读取风格文档(如存在)

### 步骤2: 调用写作Agent

调用 `universal-writer` Agent,传递:
- 文体类型
- 章节信息
- 风格要求
- 用户指示

### 步骤3: 生成内容

根据文体类型和风格要求生成内容。

### 步骤4: 保存内容

将内容保存到主文档文件。

### 步骤5: 输出结果

```
[完成] 内容撰写完成
章节: [章节标题]
字数: [XXX]字

下一步建议:
1. 使用 writer.review 进行质量检查
2. 继续撰写下一章节
```

---

## 质量检查

- [ ] 内容符合大纲要求
- [ ] 风格一致
- [ ] 格式正确
