# writer.outline - 创建写作大纲

## 命令描述

根据项目类型和用户需求,创建或更新写作大纲。

---

## 用户输入

请提供以下信息:

1. **项目路径**: [project-path]
2. **大纲类型**: 
   - new (创建新大纲)
   - update (更新现有大纲)
3. **大纲内容**: [用户提供的大纲要点或让AI生成]

---

## 执行大纲

### 步骤1: 读取项目配置

读取 `project-config.json` 获取项目信息。

### 步骤2: 生成大纲结构

根据文体类型生成相应的大纲结构:

- **academic**: 摘要 → 引言 → 文献综述 → 方法 → 结果 → 讨论 → 结论
- **article**: 引言 → 正文(多个章节) → 总结
- **creative**: 根据具体类型(小说章节、剧本幕场、诗歌节段)
- **professional**: 根据文档类型的标准结构

### 步骤3: 保存大纲

将大纲保存为 `outline.md` 文件。

### 步骤4: 输出结果

```
[完成] 大纲创建成功
文件路径: [project-path]/outline.md

大纲结构:
- 第一章: [标题]
- 第二章: [标题]
- ...

下一步建议:
使用 writer.draft 开始撰写内容
```
