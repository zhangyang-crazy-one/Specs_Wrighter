# 质量检查清单

**版本**: v1.0  
**最后更新**: 2025-01-02

本文档提供分文体的质量检查清单,用于审查文档质量。

---

## 使用说明

1. 根据文体类型选择相应的检查清单
2. 逐项检查,在方括号中标记完成状态
3. 记录发现的问题和改进建议
4. 优先处理高优先级问题

**标记说明**:
- `[ ]` - 未检查
- `[x]` - 已通过
- `[!]` - 发现问题,需要修改

## 6. 评分标准

### 6.1 评分维度

每个检查项按以下标准评分:
- **通过**: 完全符合要求
- **部分通过**: 基本符合但有改进空间
- **未通过**: 不符合要求,需要修改

### 6.2 整体评分

根据检查结果计算整体得分:

**计算方法**:
```
总分 = (通过的高优先级项 × 3 + 通过的中优先级项 × 2 + 通过的低优先级项 × 1) / 
       (总高优先级项 × 3 + 总中优先级项 × 2 + 总低优先级项 × 1) × 10
```

**评级标准**:
- 9-10分: 优秀,无明显问题
- 7-8分: 良好,有少量小问题
- 5-6分: 合格,需要改进
- 3-4分: 不合格,问题较多
- 1-2分: 严重不合格,需要重写

## 7. 问题记录模板

### 7.1 问题记录格式

```markdown
## 问题清单

### 高优先级问题
1. [CHK-A001] 摘要缺少研究方法描述
   - 位置: 第1页,摘要部分
   - 建议: 添加研究方法的简要说明

### 中优先级问题
1. [CHK-B008] 第3段过长,影响可读性
   - 位置: 第2章第1节
   - 建议: 拆分为2-3个段落

### 低优先级问题
1. [CHK-E012] 表格未对齐
   - 位置: 第4章表4-1
   - 建议: 调整表格列宽
```

### 7.2 改进建议优先级

**高优先级** (必须修改):
- 影响内容准确性的问题
- 违反学术/专业规范的问题
- 严重影响可读性的问题

**中优先级** (建议修改):
- 影响内容完整性的问题
- 格式不规范的问题
- 影响用户体验的问题

**低优先级** (可选修改):
- 细节优化问题
- 美观性问题
- 锦上添花的改进
