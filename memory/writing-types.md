# 写作类型知识库

本文档提供各种写作类型的详细说明，帮助AI助手更好地理解和支持不同类型的写作项目。

---

## 1. 学术论文

### 1.1 研究论文 (Research Paper)

**定义和特点**:
- 基于原创研究的学术文章
- 遵循科学方法和学术规范
- 包含文献综述、方法、结果、讨论
- 需要同行评审

**结构要求**:
1. 摘要 (Abstract)
2. 引言 (Introduction)
3. 文献综述 (Literature Review)
4. 研究方法 (Methodology)
5. 研究结果 (Results)
6. 讨论 (Discussion)
7. 结论 (Conclusion)
8. 参考文献 (References)

**风格规范**:
- 客观、严谨的学术语言
- 第三人称叙述
- 避免主观判断和情感表达
- 使用专业术语
- 引用格式：APA、MLA、Chicago等

**常见模式**:
- 实证研究：数据收集→分析→结论
- 理论研究：文献分析→理论构建→验证
- 案例研究：案例描述→分析→启示

### 1.2 综述论文 (Review Paper)

**定义和特点**:
- 系统性回顾某一领域的研究现状
- 不包含原创实验数据
- 综合分析现有文献
- 提出研究方向和建议

**结构要求**:
1. 摘要
2. 引言
3. 文献检索方法
4. 主题分类综述
5. 研究趋势分析
6. 研究空白识别
7. 未来研究方向
8. 结论
9. 参考文献

**风格规范**:
- 系统性和全面性
- 批判性分析
- 客观评价
- 逻辑清晰的分类

### 1.3 学位论文 (Thesis/Dissertation)

**定义和特点**:
- 硕士或博士学位要求的学术作品
- 篇幅较长（硕士3-5万字，博士8-15万字）
- 需要原创性贡献
- 严格的格式要求

**结构要求**:
1. 封面和声明
2. 摘要（中英文）
3. 目录
4. 第一章：绪论
5. 第二章：文献综述
6. 第三章：研究方法
7. 第四-N章：研究内容
8. 最后一章：结论与展望
9. 参考文献
10. 附录
11. 致谢

**风格规范**:
- 严格遵守学校格式要求
- 章节编号规范
- 图表编号和引用规范
- 公式编号规范

## 2. 研究报告

### 2.1 技术报告 (Technical Report)

**定义和特点**:
- 描述技术项目或研究的详细报告
- 面向专业读者
- 包含技术细节和数据
- 可能包含商业机密

**结构要求**:
1. 执行摘要
2. 项目背景
3. 技术方案
4. 实施过程
5. 测试结果
6. 问题和解决方案
7. 结论和建议
8. 附录（技术文档、代码等）

**风格规范**:
- 技术准确性
- 详细的技术描述
- 图表和数据可视化
- 专业术语使用

### 2.2 调研报告 (Research Report)

**定义和特点**:
- 基于调查研究的报告
- 包含数据收集和分析
- 提供决策建议
- 面向管理层或客户

**结构要求**:
1. 执行摘要
2. 调研背景和目的
3. 调研方法
4. 数据收集
5. 数据分析
6. 发现和洞察
7. 建议和行动计划
8. 附录（问卷、原始数据等）

**风格规范**:
- 清晰的数据呈现
- 客观的分析
- 可操作的建议
- 商业语言

## 3. 小说

### 3.1 长篇小说 (Novel)

**定义和特点**:
- 篇幅较长（通常8万字以上）
- 复杂的情节和人物
- 多条故事线
- 深入的主题探讨

**结构要求**:
- 三幕结构或五幕结构
- 多个章节（通常20-50章）
- 起承转合清晰
- 高潮和结局

**风格规范**:
- 统一的叙事视角
- 一致的语言风格
- 丰富的描写
- 对话和叙述平衡

**常见模式**:
- 英雄之旅：召唤→试炼→归来
- 成长故事：无知→经历→成熟
- 悬疑推理：谜题→调查→揭秘

### 3.2 中篇小说 (Novella)

**定义和特点**:
- 篇幅中等（2-8万字）
- 单一或少数故事线
- 聚焦核心冲突
- 紧凑的叙事

**结构要求**:
- 简化的三幕结构
- 10-20章
- 快速进入主题
- 集中的高潮

### 3.3 短篇小说 (Short Story)

**定义和特点**:
- 篇幅较短（通常1万字以内）
- 单一故事线
- 聚焦单一事件或主题
- 精炼的叙事

**结构要求**:
- 开端→发展→高潮→结局
- 通常无章节划分
- 快速切入
- 意外结局或开放式结局

## 4. 人物传记

### 4.1 自传 (Autobiography)

**定义和特点**:
- 作者讲述自己的人生故事
- 第一人称叙述
- 真实性要求
- 个人视角和感受

**结构要求**:
1. 童年和成长
2. 重要转折点
3. 职业发展
4. 个人关系
5. 成就和挫折
6. 反思和感悟

**风格规范**:
- 真诚的个人表达
- 时间顺序或主题顺序
- 细节和情感
- 反思性叙述

### 4.2 他传 (Biography)

**定义和特点**:
- 作者讲述他人的人生故事
- 第三人称叙述
- 基于研究和采访
- 客观性和准确性

**结构要求**:
1. 早年生活
2. 教育和成长
3. 职业生涯
4. 重大事件
5. 影响和贡献
6. 晚年和遗产

**风格规范**:
- 客观的叙述
- 详实的资料
- 平衡的评价
- 历史背景

## 5. 长篇诗歌

### 5.1 史诗 (Epic)

**定义和特点**:
- 叙述英雄事迹的长篇诗歌
- 宏大的主题和场景
- 神话或历史背景
- 韵律和格律

**结构要求**:
- 多个篇章或卷
- 英雄的旅程
- 神话元素
- 史诗性语言

**风格规范**:
- 庄严的语言
- 重复和排比
- 比喻和象征
- 韵律和节奏

### 5.2 叙事诗 (Narrative Poem)

**定义和特点**:
- 讲述故事的诗歌
- 情节和人物
- 诗歌形式
- 抒情和叙事结合

**结构要求**:
- 开端→发展→高潮→结局
- 诗节划分
- 韵律模式

---

## 6. 剧本

### 6.1 话剧剧本 (Stage Play)

**定义和特点**:
- 为舞台表演创作的剧本
- 对话为主
- 舞台指示
- 幕和场的划分

**结构要求**:
1. 人物表
2. 第一幕：开端
3. 第二幕：发展
4. 第三幕：高潮和结局
5. 舞台指示

**风格规范**:
- 对话驱动
- 舞台可行性
- 时间和空间限制
- 戏剧冲突

### 6.2 电影剧本 (Screenplay)

**定义和特点**:
- 为电影创作的剧本
- 场景描述和对话
- 镜头语言
- 标准格式

**结构要求**:
- 场景标题（INT./EXT.）
- 场景描述
- 人物对话
- 动作描述
- 转场指示

**风格规范**:
- 视觉化描述
- 简洁的对话
- 镜头感
- 标准格式（Courier字体，特定边距）

### 6.3 电视剧本 (TV Script)

**定义和特点**:
- 为电视剧创作的剧本
- 分集结构
- 广告位考虑（传统电视）
- 系列连贯性

**结构要求**:
- 冷开场（Teaser）
- 第一幕
- 第二幕
- 第三幕
- 结尾（Tag）

---

## 7. 歌剧

### 7.1 歌剧剧本 (Opera Libretto)

**定义和特点**:
- 为歌剧创作的歌词和剧本
- 配合音乐
- 诗歌化语言
- 情感表达

**结构要求**:
- 序曲
- 多个幕
- 咏叹调（Aria）
- 宣叙调（Recitative）
- 合唱

**风格规范**:
- 诗歌化语言
- 韵律和节奏
- 情感强度
- 音乐性

### 7.2 音乐剧 (Musical)

**定义和特点**:
- 结合对话、歌曲、舞蹈的剧本
- 流行音乐风格
- 现代主题
- 娱乐性

**结构要求**:
- 开场曲
- 对话场景
- 歌曲场景
- 舞蹈场景
- 终曲

**风格规范**:
- 对话和歌词结合
- 流行语言
- 节奏感
- 娱乐性

---

## 使用建议

1. **类型识别**: 在阶段一（调研）时，明确确定写作类型
2. **模板选择**: 根据写作类型选择合适的模板和结构
3. **风格适配**: 确保语言风格符合类型要求
4. **格式规范**: 遵守该类型的格式规范
5. **质量标准**: 参考该类型的质量标准进行审查
