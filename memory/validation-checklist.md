# 合规性审查检查清单

本文档提供详细的合规性审查检查清单，用于阶段五的大纲验证和最终作品审查。

---

## 检查维度概览

合规性审查包含四大维度：

1. **主题一致性检查** - 确保核心主题贯穿全文
2. **风格统一性检查** - 确保叙事风格保持一致
3. **逻辑连贯性检查** - 确保情节发展合理
4. **架构完整性检查** - 确保结构完整清晰

---

## 一、主题一致性检查

### 1.1 核心主题贯穿性

**检查标准**:
- [ ] 每个章节都与至少一个核心主题相关
- [ ] 核心主题在关键章节得到强化
- [ ] 没有偏离核心主题的章节
- [ ] 主题发展有层次感

**检查方法**:
1. 列出宪法文件中的所有核心主题
2. 逐章检查与核心主题的关联
3. 标记与主题关联薄弱的章节
4. 评估主题发展的层次

**问题识别**:
- **严重问题**: 某章节完全偏离核心主题
- **轻微问题**: 某章节与主题关联不够紧密

**示例**:
```
核心主题：成长、友谊、勇气

第1章：主角的童年（成长✓）
第2章：遇见朋友（友谊✓）
第3章：面对挑战（勇气✓）
第4章：日常生活（❌ 与主题关联薄弱）
```

### 1.2 价值观一致性

**检查标准**:
- [ ] 作品传达的价值观与宪法文件一致
- [ ] 没有与核心价值观矛盾的内容
- [ ] 价值观表达清晰明确
- [ ] 避免价值观混乱

**检查方法**:
1. 列出宪法文件中的核心价值观
2. 检查每个章节传达的价值观
3. 识别矛盾或混乱的价值观表达
4. 评估价值观的清晰度

**问题识别**:
- **严重问题**: 价值观与宪法文件直接矛盾
- **轻微问题**: 价值观表达不够清晰

**示例**:
```
核心价值观：诚实、正义、善良

第5章：主角撒谎获利（❌ 与"诚实"矛盾）
第6章：主角见义勇为（正义✓）
第7章：主角帮助他人（善良✓）
```

### 1.3 主题深化

**检查标准**:
- [ ] 主题随情节发展而深化
- [ ] 避免主题停留在表面
- [ ] 通过冲突和转折深化主题
- [ ] 结局呼应并升华主题

**检查方法**:
1. 分析主题在不同阶段的表现
2. 评估主题深化的层次
3. 检查是否有深化主题的关键场景

---

## 二、风格统一性检查

### 2.1 叙事视角一致性

**检查标准**:
- [ ] 全文使用统一的叙事视角
- [ ] 如果切换视角，有明确标识
- [ ] 视角切换符合规则
- [ ] 避免视角混乱

**检查方法**:
1. 确认宪法文件规定的叙事视角
2. 逐章检查叙事视角
3. 标记视角不一致的段落
4. 评估视角切换的合理性

**问题识别**:
- **严重问题**: 在同一章节内视角混乱
- **轻微问题**: 视角切换缺少明确标识

**示例**:
```
规定视角：第三人称限知（主角视角）

第8章：
"他看着窗外的雨，心中充满忧伤。"（✓ 正确）
"她在房间里想着他，不知道他此刻在想什么。"（❌ 错误，超出主角视角）
```

### 2.2 语言风格一致性

**检查标准**:
- [ ] 语言风格符合宪法文件要求
- [ ] 全文保持统一的语言风格
- [ ] 对话风格符合人物设定
- [ ] 避免风格突变

**检查方法**:
1. 确认宪法文件规定的语言风格
2. 抽查各章节的语言风格
3. 检查对话的风格一致性
4. 识别风格突变的段落

**问题识别**:
- **严重问题**: 语言风格与宪法文件完全不符
- **轻微问题**: 个别段落风格略有偏差

**示例**:
```
规定风格：正式、文学化

第9章：
"夕阳西下，晚霞如血。"（✓ 符合）
"太阳下山了，天空红红的。"（❌ 过于口语化）
```

### 2.3 修辞手法一致性

**检查标准**:
- [ ] 修辞手法符合宪法文件要求
- [ ] 避免使用禁止的修辞手法
- [ ] 修辞手法使用适度
- [ ] 修辞风格统一

**检查方法**:
1. 列出宪法文件中允许/谨慎/禁止的修辞手法
2. 检查各章节的修辞使用
3. 标记不符合要求的修辞
4. 评估修辞使用的适度性

**问题识别**:
- **严重问题**: 使用了明确禁止的修辞手法
- **轻微问题**: 修辞手法使用过度或不足

### 2.4 描写手法一致性

**检查标准**:
- [ ] 描写手法符合宪法文件要求
- [ ] 主要描写方式保持一致
- [ ] 描写详略得当
- [ ] 避免描写风格突变

**检查方法**:
1. 确认宪法文件规定的主要描写手法
2. 检查各章节的描写方式
3. 评估描写的详略程度
4. 识别描写风格不一致的段落

---

## 三、逻辑连贯性检查

### 3.1 因果关系清晰性

**检查标准**:
- [ ] 重大事件有明确的因果关系
- [ ] 避免突兀的情节转折
- [ ] 因果链条完整
- [ ] 动机和结果合理

**检查方法**:
1. 列出所有重大事件
2. 分析每个事件的原因和结果
3. 绘制因果关系图
4. 识别因果关系薄弱的环节

**问题识别**:
- **严重问题**: 重大事件缺少合理原因
- **轻微问题**: 次要情节的因果关系不够清晰

**示例**:
```
事件：主角突然决定离开家乡

原因分析：
- 是否有足够的动机？（✓/❌）
- 动机是否在之前章节铺垫？（✓/❌）
- 决定是否符合人物性格？（✓/❌）
```

### 3.2 时间线一致性

**检查标准**:
- [ ] 时间线清晰明确
- [ ] 没有时间矛盾
- [ ] 时间跨度合理
- [ ] 时间标记准确

**检查方法**:
1. 绘制完整的时间线
2. 标记每个章节的时间点
3. 检查时间跨度的合理性
4. 识别时间矛盾

**问题识别**:
- **严重问题**: 时间线出现明显矛盾
- **轻微问题**: 时间标记不够清晰

**示例**:
```
时间线检查：

第1章：2020年春天
第2章：2020年夏天（✓ 合理）
第3章：2019年秋天（❌ 时间倒流，矛盾）
```

### 3.3 角色行为逻辑性

**检查标准**:
- [ ] 角色行为符合性格设定
- [ ] 角色成长有合理过程
- [ ] 避免角色行为突变
- [ ] 角色动机清晰

**检查方法**:
1. 列出所有主要角色的性格设定
2. 检查每个角色的行为是否符合设定
3. 分析角色成长的合理性
4. 识别行为不合逻辑的情节

**问题识别**:
- **严重问题**: 角色行为与设定完全矛盾
- **轻微问题**: 角色行为缺少足够动机

**示例**:
```
角色：李明（性格：谨慎、保守）

第10章：李明冲动地做出重大决定（❌ 与性格矛盾）

修改建议：
- 增加促使李明改变的重大事件
- 或修改决定的性质，使其符合谨慎性格
```

### 3.4 世界观一致性

**检查标准**:
- [ ] 世界规则保持一致
- [ ] 没有违反世界观设定的情节
- [ ] 魔法/科技系统逻辑自洽
- [ ] 社会规则一致

**检查方法**:
1. 列出所有世界观设定
2. 检查每个章节是否遵守设定
3. 识别违反世界观的情节
4. 评估世界观的自洽性

---

## 四、架构完整性检查

### 4.1 章节结构完整性

**检查标准**:
- [ ] 所有必需章节齐全
- [ ] 章节顺序合理
- [ ] 章节长度适当
- [ ] 章节划分清晰

**检查方法**:
1. 对照大纲检查章节完整性
2. 评估章节顺序的合理性
3. 检查章节长度的平衡性
4. 评估章节划分的清晰度

**问题识别**:
- **严重问题**: 缺少关键章节
- **轻微问题**: 章节长度不够平衡

### 4.2 起承转合清晰性

**检查标准**:
- [ ] 开端引人入胜
- [ ] 发展层层推进
- [ ] 高潮震撼人心
- [ ] 结局令人满意

**检查方法**:
1. 识别起承转合的关键章节
2. 评估每个部分的效果
3. 检查节奏的合理性
4. 评估整体结构的完整性

**问题识别**:
- **严重问题**: 缺少高潮或结局
- **轻微问题**: 节奏控制不够理想

### 4.3 伏笔回收完整性

**检查标准**:
- [ ] 所有伏笔都有回收
- [ ] 回收时机合理
- [ ] 没有多余的伏笔
- [ ] 伏笔与回收呼应

**检查方法**:
1. 列出所有伏笔
2. 检查每个伏笔的回收情况
3. 评估回收时机的合理性
4. 识别未回收或多余的伏笔

**问题识别**:
- **严重问题**: 重要伏笔未回收
- **轻微问题**: 回收时机不够理想

**示例**:
```
伏笔清单：

第2章：神秘的项链（伏笔）
第15章：项链的秘密揭晓（回收✓）

第5章：陌生人的警告（伏笔）
未回收（❌ 严重问题）
```

### 4.4 主题呼应

**检查标准**:
- [ ] 结局呼应开头
- [ ] 主题得到升华
- [ ] 核心问题得到解答
- [ ] 留有适当余韵

**检查方法**:
1. 对比开头和结尾
2. 评估主题的升华程度
3. 检查核心问题的解答
4. 评估结局的满意度

---

## 使用建议

1. **系统检查**: 按照四大维度系统性地进行检查
2. **记录问题**: 详细记录发现的所有问题
3. **分类处理**: 区分严重问题和轻微问题，优先处理严重问题
4. **多次审查**: 修改后重新审查，确保问题得到解决
5. **客观评价**: 基于宪法文件的明确要求，避免主观判断
6. **完整记录**: 在验证报告中完整记录检查过程和结果
