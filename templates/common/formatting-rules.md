# 格式规范 (Formatting Rules)

**适用场景**: 统一文档格式标准  
**创建日期**: [DATE]

---

## 1. Markdown格式规范

### 1.1 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 1.2 列表

**无序列表**:
```markdown
- 项目1
- 项目2
```

**有序列表**:
```markdown
1. 项目1
2. 项目2
```

### 1.3 代码块

```markdown
​```语言
代码内容
​```
```

### 1.4 表格

```markdown
| 列1 | 列2 |
|-----|-----|
| 内容1 | 内容2 |
```

---

## 2. 引用格式

### 2.1 行内引用

[作者, 年份]

### 2.2 块引用

> 引用内容
> 
> —— 作者

---

## 3. 图片格式

```markdown
![图片描述](图片路径)
```

---

## 4. 链接格式

```markdown
[链接文本](URL)
```
